Skeleton Annotation

My bachelor thesis was centered around creating a customizable pipeline for annotating human skeletons in image datasets. This tool is created for researchers and developers working in computer vision, particularly in pose estimation and person/object detection. The estimation of keypoints on images could for instance be used to predict the direction and speed of walking or to predict if a person has seen the vehicle. Futhermore it could be used to create 3D meshes of the skeleton models for augmented and virtual reality. It integrates seamlessly with MMDetection, MMPose and LabelMe. The final pipeline includes the automated and adjustable process, anonimyzing people in the dataset and different export options, also for manual adjustments.

Mobile Robot Control

This project involved the creation of a person detection and robot control package within ROS2. The objective of the robot is to move through the path and come to a complete stop in front of the human. The creation of these two packages was done within Ubuntu.
